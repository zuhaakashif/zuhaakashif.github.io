AcuCare — Mobile Acne Analysis

AcuCare is an applied computer vision project focused on lightweight, interpretable acne analysis for mobile devices. The pipeline emphasizes efficiency, transparency, and practical deployment rather than heavy deep learning models.

← Back View Code

Motivation

Many dermatology-focused mobile applications rely on large neural networks that are difficult to interpret and deploy on low-resource devices. AcuCare explores whether classical image processing and lightweight machine learning can provide usable performance while remaining interpretable and efficient.

Pipeline Overview

The pipeline consists of image preprocessing, lesion segmentation, handcrafted feature extraction, and acne severity classification. Each stage is designed to be explainable and computationally inexpensive.

Methods

Segmentation is performed using classical thresholding and morphological operations. Texture and color features are extracted to represent lesion characteristics. A lightweight classifier is used for severity prediction.

Results

The system achieved an accuracy of 87.7% on a held-out test set, demonstrating that interpretable pipelines can remain competitive for constrained deployment scenarios.

Why This Matters

This project complements my research on AI reliability by emphasizing transparency and failure awareness. While AcuCare is not a safety-critical system, its design philosophy aligns with broader goals of responsible and interpretable medical AI.

Key Takeaways

  • Lightweight pipelines can perform competitively.
  • Interpretability aids debugging and trust.
  • Mobile constraints require design trade-offs.
  • Classical methods remain relevant in applied settings.
